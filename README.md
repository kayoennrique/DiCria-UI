## 👨‍💻 How to install?

```bash
# Using  npm
npm i dicria-ui

# Using  yarn
yarn add dicria-ui
```

## ⌨️ Scripts

- Clone the repository:

```bash
git clone https://github.com/kayoennrique/DiCria-UI-1.git
```

- Navigate to the project directory:

```bash
cd DiCria-UI-1
```

- Install dependencies:

```bash
npm install
```
